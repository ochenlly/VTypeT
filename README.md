## 关于本项目

`VTypeT` 是一个后台管理系统的案例，免费且开源

### 主要功能

1. 登录
2. 注册

### 功能特性

- ✔ 简单明了
- 👍 上手快

## 可发者须知

### 技术框架

- 数据库：MYSQL
- 后端框架：Express
- 前端框架：Vue

### 开发环境

Windows 10 with vscode

## Build Setup

```
# install dependencies
pnpm i

# serve with hot reload at localhost:8080
pnpm dev

# build for production with minification
pnpm build
```
